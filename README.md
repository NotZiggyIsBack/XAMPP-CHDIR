# XAMPP-CHDIR
```
⚙️ XAMPP-CHDIR v1.0

This script allows you to quickly change XAMPP working directory aka root folder of xampp project.
Made this for myself so I can move between projects quicker but why not release yes :)

How to use:
1. Download this repo
2. run "start.bat"
3. Enjoy! 🎉

It will auto download requirement aka colorama package when you run start.bat
```
